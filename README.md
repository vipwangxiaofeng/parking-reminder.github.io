# 停车侠 - 精准提醒，拒绝被坑

一个简洁实用的停车提醒应用，帮助用户避免停车超时罚款。

## 功能特点

- **精准计时**：记录实际停车时间，计算停车费用
- **自定义规则**：支持多种计费规则设置
- **提前提醒**：在计费周期结束前发出提醒
- **历史记录**：保存停车记录，方便查询
- **移动端适配**：优化手机显示和触摸体验
- **浏览器通知**：支持桌面和移动端通知提醒
- **PWA支持**：可添加到主屏幕，离线使用

## 移动端使用说明

### 1. 添加到主屏幕

#### iOS (Safari)
1. 打开应用
2. 点击底部的"分享"按钮
3. 选择"添加到主屏幕"
4. 点击"添加"

#### Android (Chrome)
1. 打开应用
2. 点击右上角的"菜单"按钮
3. 选择"添加到主屏幕"或"安装应用"
4. 点击"添加"

### 2. 通知设置

应用需要通知权限才能在停车时间即将结束时提醒您：

1. 首次打开应用时，会请求通知权限，请点击"允许"
2. 如果不小心拒绝了，可以在手机设置中重新开启：
   - iOS: 设置 > 通知 > 停车侠 > 允许通知
   - Android: 设置 > 应用 > 停车侠 > 通知 > 开启通知

### 3. 离线使用

应用支持离线使用，即使没有网络连接也能正常计时和提醒。

## 常见问题

### Q: 为什么通知不工作？
A: 请检查：
1. 是否授予了通知权限
2. 手机是否处于勿扰模式
3. 应用是否在后台运行

### Q: 如何更新应用？
A: 应用会自动检查更新，您也可以通过刷新页面手动更新。

### Q: 数据是否安全？
A: 所有数据都存储在您的设备本地，不会上传到服务器。
